# Example Landmark — 建筑需求示例

这是给 AI 的项目内需求文件。实际协作时先写清楚本文件，再要求 AI 只修改 `main.py` 与 `src/`。

## 核心概念

- 建筑主题：自定义维度中的石质传送地标。
- 第一眼轮廓：四座角塔围绕中央大厅，中央灯柱高于屋顶。
- 用途：演示大型结构的全自动世界生成和小型结构的 ModSDK 放置。

## 空间与路线

- 入口朝向负 Z，推荐传送点位于入口外。
- 中央大厅入口保持敞开，十字灯带引导玩家进入中心。
- 四座塔用于检查跨切片接缝是否准确。

## 风格与材料

- 主材：石砖、平滑石。
- 强调材：磨制黑石砖、錾制石砖、海晶灯。
- 玻璃开窗提供方向提示。

## 验收条件

- [x] 返回尺寸严格为 `96×48×96` 的一个逻辑结构。
- [x] 推荐传送点正对负 Z 入口。
- [x] 可同时生成 worldgen 与 ModSDK 两套资源。
- [x] 所有切片不超过 65,536 方块。
